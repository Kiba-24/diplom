# Инструкция оператора
- Войти в интерфейс кнопкой "Войти оператором".
- Загрузить список обращений.
- Создать обращение с темой/описанием/категорией.
- Отправить сообщение боту для первичной классификации.
- Ошибка 401: повторить вход и проверить токен.
- Ошибка 400: проверить обязательные поля формы.
